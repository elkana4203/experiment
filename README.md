# Minimal Python Chess

This workspace contains a minimal, text-based chess game in Python.

Run the interactive CLI:

```bash
python game.py
```

Run the demo sequence:

```bash
python game.py --demo
```

Moves use coordinate format like `e2e4`.

Helpers included:

- `check_python.ps1`: diagnose Python install and PATH issues.
- `setup_venv.ps1`: create a virtual environment (`.venv`) and show activation command.
- `run_demo.ps1`: finds `python` or `py` and runs the demo (`game.py --demo`).

PowerShell usage examples:

```powershell
.\check_python.ps1
.\setup_venv.ps1
.\run_demo.ps1
```
